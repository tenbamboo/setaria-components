# 安装

## 使用包管理器

```shell
# 选择一个你喜欢的包管理器

# NPM
$ npm install setaria-components --save

# Yarn
$ yarn add setaria-components

# pnpm
$ pnpm install setaria-components
```

## 安装依赖项

```shell
# 选择一个你喜欢的包管理器

# NPM
$ npm install vxe-table xe-utils element-plus @element-plus/icons-vue --save

# Yarn
$ yarn add vxe-table xe-utils element-plus @element-plus/icons-vue

# pnpm
$ pnpm install vxe-table xe-utils element-plus @element-plus/icons-vue
```
